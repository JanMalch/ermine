Ermine

Easily explore any GitHub repository in an IDE look.

Features

Ermine is a simple web app that allows you to browse any GitHub repository in an IDE look. You can easily open multiple files with syntax highlighting for a huge amount of languages, or view images and CSV tables! Over 190 languages are supported for highlighting, with the help of Prism.js.
