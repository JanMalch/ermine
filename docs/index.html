Ermine

Ermine is a simple web app that allows you to browse any GitHub repository in an IDE look. You can easily open multiple files with syntax highlighting for a huge amount of languages, or view images and CSV tables! Over 190 languages are supported for highlighting, with the help of Prism.js.

Check out the source code.

How it's build

Ermine is built with Angular 7, Angular Material and GitHub API v4.

The development process makes use of various Continuous Integration providers.

Provider Status Tasks
Travis CI Runs linter, tests and deploy
Code Climate Ensures highest code quality
snyk Checks dependencies for vulnerabilities
