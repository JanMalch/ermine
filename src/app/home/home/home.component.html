WELCOME TO ERMINE

Ermine is a simple web app that allows you to browse any GitHub repository in an IDE look.

Select a repository

You can easily open multiple files with syntax highlighting for a huge amount of languages, or view images and CSV tables!

{{ languages.length }} languages are supported!
  • {{ lang.name }}

Head over to the select page, login with GitHub, and type in the owner and name of the repository you want.
You can also type in any branch, commit hash, or open the dropdown and fetch all available branches.

For bonus convenience add this bookmarklet to open the GitHub repository in your current tab here on Ermine! 

javascript:window.open("https://ermine.netlify.com/view?url=" + location.pathname, "_blank");

Sometimes Ermine can't properly parse the URL, for example when there are slashes in the branch name. You will be redirected to the select page, where you can fix the values.
