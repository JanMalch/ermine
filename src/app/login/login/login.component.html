A login is required to use Ermine. Please login with your GitHub account.

You can choose to login with either the basic scope or the extended scope. With the basic scope you will be able to read all public repositories. Only with the extended scope you will be able to read private repositories that you have access to, including your very own ones.

You can always completely revoke the access in your GitHub settings.
